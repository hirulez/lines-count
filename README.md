# simple-lines-count README

Show selected lines count in a status bar with the ability to specify output format.

Default format is:
Lns: {Total} ({Not empty} + {Empty}e + {Comments}c)

e.g. Lns: 10 (5 + 3e + 2c) 
5 not empty
3 empty
2 comments
